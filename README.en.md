# 🛍️ EcoTote Project — Responsive Promo Website

## 📄 Project Description

**EcoTote Project** is a promotional website for an eco-friendly tote bag brand
focused on style, practicality, and responsible consumption. The site showcases
the product assortment, key advantages, a photo gallery, customer feedback, and
a contact section. The project is fully adapted for mobile, tablet, and desktop
screens.

## 🔗 Links

- [GitHub Repository](https://github.com/vitaliifedunyk/eco-tote-project)
- [Live Demo](https://vitaliifedunyk.github.io/eco-tote-project/)

## 🧩 Features

- Responsive mobile-first layout for `320px+`, `768px+`, and `1440px+`.
- Multi-section one-page structure: Hero, About, Advantages, Assortment,
  Gallery, Feedbacks, Support, Footer.
- Mobile burger menu with open/close behavior and background scroll lock.
- Smooth anchor navigation between sections.
- Product catalog with 8 items including title, description, price, and buy
  button.
- Contact form with basic native HTML field validation.
- Responsive images implemented with `<picture>` and `srcset`.
- Contact details and social links in the header and footer.

## 🛠 Tech Stack

- HTML5
- CSS3 (PostCSS)
- JavaScript (ES Modules)
- Vite
